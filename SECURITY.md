# セキュリティポリシー

## 脆弱性の報告

本プロジェクトでセキュリティ上の脆弱性を発見された場合は、以下の手順で報告をお願いします。

### 報告方法

1. **GitHub Security Advisory**（推奨）: [Security Advisory を作成](https://github.com/zeimu-ai/open-journal-rules/security/advisories/new)してください
2. **メール**: info@ma-navigator.com 宛に「[SECURITY] open-journal-rules」の件名でご連絡ください

### お願い

- 公開Issueでの脆弱性報告は**避けてください**（修正完了前の情報公開を防ぐため）
- 報告には再現手順をできるだけ詳しく記載してください
- 報告後、48時間以内に受領確認をお送りします

### 対象範囲

- 仕訳ルール定義ファイルの不正なデータ注入
- テスト・バリデーションロジックの脆弱性
- 依存パッケージの脆弱性

### 対象外

- 仕訳ルールの税務的な正確性に関する問題 → [ルールの誤り報告](https://github.com/zeimu-ai/open-journal-rules/issues/new?template=bug_report.yml)をご利用ください
